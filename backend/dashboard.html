🎯 ¿Qué es el Trade Room?
El Trade Room ejecuta apuestas reales en Polymarket usando tu capital configurado.

Cómo usarlo:
1. Asegúrate de que tu wallet de Polymarket está conectada en Configuración
2. El escáner de mercados detecta automáticamente oportunidades con edge > 5%
3. Revisa cada señal: verás el mercado, la dirección (YES/NO), el monto y el edge
4. Haz clic en EJECUTAR para colocar la apuesta, o IGNORAR para descartarla
5. Modo DRY RUN = simulación sin dinero real. Desactívalo en Configuración para operar en vivo

⚠️ Siempre opera en DRY RUN primero para validar la estrategia.
